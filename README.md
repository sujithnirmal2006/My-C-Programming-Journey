## 🚀 Getting Started

### Clone the Repository
```bash
git clone https://github.com/sujithnirmal2006/My-C-Programming-Journey.git
cd My-C-Programming-Journey
```

### Compile and Run Any Program
```bash
cd Assessment_1
gcc question_1.c -o question_1
./question_1
```

### View the Complete Catalog
Open `COMPLETE_CODE_CATALOG.md` to see:
- Index of all 186+ programs
- Problem descriptions
- Difficulty levels
- Topics covered
